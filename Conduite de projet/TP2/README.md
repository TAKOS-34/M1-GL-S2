Il y a deux versions de notre rendu.

# V1

La première version (dossier `v1`) correspond à la première approche qui nous est venue à l'esprit : un gros prompt général, rédigé par nos soins, pour commencer la discussion.

# V2

Pour la deuxième version (dossier `v2`), nous avons essayé une autre approche. Nous avons donné le sujet à ChatGPT et lui avons demandé de rédiger un prompt que nous avons ensuite utilisé comme point de départ pour notre travail. Nous n'avons pas fourni le reste des prompts, seulement le premier, car la discussion est à peu près la même et présente donc peu d'intérêt.

# Conclusion

Le code donné est à peu près le même pour un nombre identique de prompts, il n'y a donc pas de grande différence. Peut-être que cette méthode (demander à une IA de rédiger le prompt) sera plus efficace dans le futur, quand les IA seront plus performantes.
